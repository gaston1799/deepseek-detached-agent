DeepSeek terminal agent

Watch DeepSeek think, call tools, and work inside your repo.

dsw is a Windows-friendly CLI for interactive DeepSeek coding sessions, permissioned file and shell tools, resumable session memory, and detached Markdown subagents.

View source Install CLI docs

Quick install

irm https://raw.githubusercontent.com/gaston1799/deepseek-detached-agent/master/install.ps1 | iex

Commands: dsw for interactive sessions, dsd for detached jobs, and dswait for quiet output polling.

Features

Built for agent workflows that need receipts.

Interactive

Streaming thinking

Run dsw to open a dashboard, choose permissions, stream model reasoning, and continue chatting until you exit.

Workspace tools

Read, patch, write, command

List files, read line ranges, patch exact text, write files, and run PowerShell or cmd with review, ask, or full-access modes.

Session memory

Resume context

Every session is saved under .deepseek-watch/sessions, including tool results, permissions, and model context.
